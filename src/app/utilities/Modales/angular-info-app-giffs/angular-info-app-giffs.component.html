Conocimientos adquiridos

Comunicación entre componentes

  • Se utiliza el decorador input para pasar datos desde un componente padre a un componente hijo.

  • Se utiliza el decorador output se utiliza para emitir eventos desde un componente hijo hacia un componente padre.

Peticiones http

  • Se crea un servicio para realizar peticiones http a un Api externa o Apis propias.

  • Se consume el servicio para suscribirnos al método que trae la información del Api y por parámetro se le manda la palabra que se desea buscar en el api de GHYPI.
